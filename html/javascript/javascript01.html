01. 변수 : 데이터 저장

변수는 데이터를 저장하는 저장소입니다
var는 동일한 변수를 중복해서 사용이 가능합니다. 

{
    var x = 100;                //변수 x에 숫자 100을 저장함
    var y = 200;                //변수 y에 숫자 200을 저장함
    var z = "javascript";       //변수 z에 문자 "javascript"를 저장함
    
    console.log(x);
    console.log(y);
    console.log(z);
}
100
200
javascript

02. 변수 : 데이터 저장 + 데이터 변경

변수는 데이터를 저장할수도 있지만 변경도 가능합니다.
let은 이미 선언한 변수를 중복해서 선언 할 수 없습니다. 

{
    let x = 100;
    let y = 200;
    let z = "javascript"

    x = 100;
    y = 200;
    z = "react"

    console.log(x);
    console.log(y);
    console.log(z);
}
100
200

03. 변수 : 데이터 저장 + 데이터 변경 + 데이터 추가

변수는 데이터를 저장하고, 변경하고, 추가할 수 있습니다.

 {
    let x = "100";
    let y = 200;
    let z = "javascript"

    x += 300; // x = x + 300
    y += 400;
    z += "react"

    console.log(x);
    console.log(y);
    console.log(z);
}

05. 배열 : 데이터 저장(여러개) : 표현방법1

배열은 데이터를 여러개 저장 할 수 있습니다.
배열이 생성되면 크기는 임의로 변경이 가능하고 이미 할당된 값은 사라지지 않습니다. 

{
    const arr = new Array();
    arr[0] = 100;
    arr[1] = 200;
    arr[2] = "javascript";

    console.log(arr[0]);
    console.log(arr[1]);
    console.log(arr[2]);
}
100
200
